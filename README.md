<h1 align="center" id="title">Apps GPL Library</h1>

<p id="description">A library of helpful functions that can be quickly linked in any GPL project. Requires GDS Installed</p>

<p align="center"><img src="https://img.shields.io/badge/github-Builds-blue?style=for-the-badge&amp;logo=github&amp;link=https%3A%2F%2Fgithub.com%2Fprecise-automation%2FApps-GPL-Library%2Freleases" alt="shields"></p>

<h2>🛠️ Installation Steps:</h2>

<p>1. Import Project</p>

<p>2. On the target main project right-click &gt; Edit Project Properties</p>

<p>3. In the Project Properties window Click "Add"</p>

<p>4. In the Project Dropdown select the Lib_Apps project and click "Accept"</p>

<p>5. Click "Accept" in the Project Properties window</p>
