# Apps-GPL-Library
A library of helpful functions that can be quickly linked in any GPL project

To Implement:
1. Import Project
2. On the target main project, right-click > Edit Project Properties
3. in the Project Properties window, Click "Add"
4. In the Project Dropdown select the Lib_Apps project and click "Accept"
5. Click "Accept" in the Project Properties window

Done, now you can make calls to any of the functions in Apps Library. Any time you drag the main project into the controller, GDS will also import the Apps Library.
